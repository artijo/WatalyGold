# WatalyGold Application

แอปพลิเคชั่นวิเคราะห์คุณภาพมะม่วงน้ำดอกไม้สีทอง โดยใช้การประมวลผลภาพ ซึ่งจะช่วยให้ผู้ใช้งานทราบว่ามะม่วงลูกนี้ มีคุณภาพอยู่ในระดับใด โดยอ้างอิงเกณฑ์ตามมาตราฐาน มกษ. ปี 2558
